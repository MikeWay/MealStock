User Guide

SCM Connection

Most tools in this dashboard work by controlling SCM on your behalf, so an active SCM connection is required before you can use them. To connect, click the Connect to SCM button in the top-right of the header.

A dialog will appear asking for your SCM username and password. After entering them, SCM will send a verification code to your mobile phone by SMS — enter that code in the second step to complete the login. Once connected, the button turns green and shows your SCM username.

If credentials have been saved, the app will attempt to connect automatically when it starts — the button flashes orange while this is in progress. If you are prompted to connect when trying to use a tool, click Connect to SCM in the header first, then try again.

Home

The home screen shows live counts for key areas — tasks, duplicates, email issues, name problems, photo consents, and failing scheduler jobs. Click any tile to jump straight to that tool. Counts update after SCM connects on startup.

Contacts › Duplicates

Scans all contacts for pairs with similar names. Definite matches share a near-identical name. Possible matches are flagged with a reason — check them carefully before acting.

  • Click Preview Merge to see which fields will be kept, then Confirm Merge to apply.
  • Click Not a Duplicate to tag the first contact and remove the pair from future scans.
  • Use Swap to change which contact is kept as the primary record.

Contacts › Name Repair

Finds contacts whose names have capitalisation or whitespace issues (e.g. "john smith", "MARY JONES"). Click Check Names to run the scan, review the list, exclude any you want to leave unchanged, then click Apply Fixes to update SCM.

Contacts › Email Issues

Shows contacts with a bounce or malformed address recorded in SCM. The Fix Email Issues tool resets the mailing-error flag on each affected contact so they will receive emails again — use this after the underlying address problem has been corrected.

Tasks & Status › SCM Tasks

Lists items in SCM's "In Need of Attention" queue — bookings, membership renewals, and courses that need a manual action. Each item links directly to the relevant SCM page.

Tasks & Status › Scheduler

Shows all background jobs registered with the scheduler, when each last ran, and whether it succeeded or failed. Failed jobs are highlighted. The scheduler runs jobs automatically at their configured times — no manual action is needed unless a job is consistently failing.

Content › Upload Minutes

Syncs a set of committee-meeting minutes from a local folder into SCM. Browse to the folder, then click Sync & Regenerate. Already-uploaded files are skipped; only new ones are added.

Content › Photo Consents

Shows each member's photo-consent status. Click Scan Consents to fetch the latest data from SCM (this may take a minute). Use the filter buttons to view all members, or just those consented, withdrawn, or without a record. You can withdraw a member's consent, or restore it, from the list. Download CSV exports the full list.

Admin › Users

Create and delete dashboard accounts, reset passwords, and assign roles. Roles control which tools each user can see and what actions they can perform. The first account is created during initial setup without requiring an existing login.

Admin › Clear Cache

The contact list is cached locally to avoid slow SCM fetches on every page load. Clicking Clear Cache deletes the cached file so the next contact operation fetches fresh data from SCM.
